### Dependencies
```
yarn install
```

### Uglified/Minified Production Output
```
node_modules/.bin/webpack -p
```
*node_modules/.bin/ ensures that the local version runs*

### Dev Server
```
node_modules/.bin/webpack-dev-server
```

### Outputs
```
dist/app.bundle.js
```
